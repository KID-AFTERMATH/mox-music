# YouTube Music Stream & Download App

A Streamlit application that allows you to stream and download audio from YouTube and YouTube Music, create playlists, and batch download as ZIP files.

## Features

- **Stream audio** directly from YouTube/YouTube Music
- **Download as MP3** with configurable quality
- **Create and manage playlists**
- **Batch download** entire playlists as ZIP files
- **Import/Export playlists** as JSON files
- **Clean interface** with progress tracking
